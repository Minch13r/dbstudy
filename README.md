## 기본 SQL 문법 ⌨️
### DDL (Data Definition Language)
- CREATE
- ALTER
- DROP
- TRUNCATE

### DML (Data Manipulation Language)
- SELECT
- INSERT
- UPDATE
- DELETE

### DCL (Data Control Language)
- GRANT
- REVOKE

- 정규화
- 인덱스 설계
- 테이블 구조 최적화
